# Pedagogisk Arkiv App

En Python-applikasjon for barnehageansatte som vil søke i akademisk faglitteratur.
Appen søker tverrfaglig innen pedagogikk, psykologi, etikk og sosiologi.

## Funksjoner
- Bredt akademisk søk via Semantic Scholar API.
- Lagring av artikler i lokalt arkiv.
- Direkte lenker til fulltekst-artikler.
- Designet for Android og PC.

## Installasjon
1. Installer Python 3.
2. Kjør `pip install -r requirements.txt`.
3. Start appen med `python main.py`.
